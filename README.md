### 🏗️ Platform engineering @ Lidl

On the Platform Team, I bridge React Native into native apps and build the shared foundations other teams build on — a design system with real parity between React and React Native, a communication layer across the JS↔native boundary, and a component catalog that saves everyone from reinventing the same button twice. BridgeKit above is that same bridge problem, tackled again in the open.
